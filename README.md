# Inventory-Management-System
A full-stack web application to manage inventory efficiently. It allows users to add, view, update, and delete products with secure authentication.

# Features
User Registration & Login
Dashboard Overview
Add, View, Update, Delete Products (CRUD)
Responsive UI
RESTful API Integration
# Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
Tools: Visual Studio Code
